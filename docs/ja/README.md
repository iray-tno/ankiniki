### 翻訳ガイドライン

日本語翻訳では以下の原則に従います：

#### 専門用語の扱い

- **技術用語**: 初出時に英語を併記（例: フラッシュカード (flashcard)）
- **固有名詞**: 英語のまま使用（Anki, AnkiConnect, Electron等）
- **一般的な用語**: 日本語に翻訳（設定→Settings, インストール→Installation）

#### 文章スタイル

- **自然な日本語**: 直訳よりも自然な日本語表現を優先
- **敬語レベル**: 丁寧語レベルで統一（です・ます調）
- **技術文書的**: 簡潔で正確な表現

#### 一貫性のある用語

| 英語          | 日本語           | 備考       |
| ------------- | ---------------- | ---------- |
| Flashcard     | フラッシュカード |            |
| Deck          | デッキ           |            |
| Study         | 学習/復習        | 文脈による |
| Configuration | 設定             |            |
| Installation  | インストール     |            |
| Tutorial      | チュートリアル   |            |
| Guide         | ガイド           |            |
| Reference     | リファレンス     |            |
