# Change Log

All notable changes to the "ankiniki-vscode" extension will be documented in this file.

## [0.1.0] - 2024-12-08

### Added

- Initial release of Ankiniki VS Code extension
- **Add Selected Text as Flashcard** command (`Ctrl+Shift+A`)
- **Add Code Block as Flashcard** command (`Ctrl+Shift+C`)
- **Quick Add Flashcard** command (`Ctrl+Shift+Q`)
- **Ankiniki Sidebar** with quick action buttons
- Right-click context menu integration
- Configurable settings for AnkiConnect URL, default deck, and model
- Automatic code language detection and syntax highlighting
- File path inclusion in generated cards
- Smart question generation based on code context
- AnkiConnect connection testing and status notifications
- Cross-platform keyboard shortcuts (Windows, Mac, Linux)

### Features

- Seamless integration with existing Anki workflow via AnkiConnect
- Support for all programming languages with syntax highlighting
- Intelligent code understanding for better card generation
- Customizable card templates and tagging system
- Real-time connection status with Anki
- Rich HTML formatting for code blocks in Anki cards

### Requirements

- VS Code 1.74.0 or higher
- Anki Desktop with AnkiConnect addon (code: 2055492159)
- AnkiConnect addon must be running on default port 8765
