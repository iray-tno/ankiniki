# Ankiniki Project Context for Claude

This file provides comprehensive context about the Ankiniki project for AI assistants.

## Project Overview

**Ankiniki** is an Anki companion tool specifically designed for software engineers and developers. It eliminates the manual effort of flashcard creation while providing modern UI/UX and seamless workflow integration.

### Current Status: Phase 1 MVP Complete ✅

The project has successfully completed its initial implementation with:
- **Backend API**: Express.js server with AnkiConnect integration
- **Desktop App**: Electron application with React frontend  
- **CLI Tool**: Command-line interface for power users
- **Shared Infrastructure**: TypeScript types and utilities

## Architecture

### Monorepo Structure
```
ankiniki/
├── packages/
│   ├── shared/          # Common types, utils, constants
│   └── backend/         # Express.js API server
├── apps/
│   ├── desktop/         # Electron desktop app
│   └── cli/             # Command-line interface
└── scripts/             # Development and build tools
```

### Technology Stack
- **Languages**: TypeScript, JavaScript
- **Frontend**: React, Electron
- **Backend**: Node.js, Express.js
- **CLI**: Commander.js, Inquirer.js
- **Quality**: ESLint, Prettier, Husky
- **Integration**: AnkiConnect API

### Key Design Principles

1. **Companion Tool Strategy**: Enhances Anki without replacing it
2. **AnkiConnect Integration**: All data operations through official API
3. **Developer-First**: Optimized for programmer workflows
4. **Type Safety**: Strict TypeScript with Zod runtime validation
5. **Cross-Platform**: Windows, macOS, Linux support

## Domain Knowledge

### Anki Concepts
- **Spaced Repetition**: Algorithm for optimal memory retention
- **Cards**: Question/answer pairs with scheduling metadata
- **Decks**: Collections of thematically related cards
- **Models**: Card templates (Basic, Cloze, etc.)
- **AnkiConnect**: Third-party addon enabling API access

### Problems Solved
1. **Manual Card Creation**: Reduces time from minutes to seconds
2. **Limited Tech Content**: Adds code syntax highlighting and markdown
3. **Workflow Fragmentation**: Integrates with developer tools
4. **Batch Processing**: Enables bulk operations

## Common Issues & Solutions

### AnkiConnect Problems
1. Verify Anki desktop is running
2. Check AnkiConnect addon installation
3. Test connectivity with health endpoint
4. Review CORS settings if needed

### Development Issues
1. Ensure Node.js 18+ installed
2. Run `npm run clean` and reinstall
3. Check workspace dependencies
4. Verify TypeScript compilation
